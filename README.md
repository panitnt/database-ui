# database-ui

## Instruction

* Clone this project
```
git clone https://github.com/panitnt/database-ui.git
```

* Get into project folder
```
cd database-ui
```

* Install module
```
npm i
```

* Create ```.env``` file (also see sample in sample.env)
```
PORT=8083
DB_SERVER=localhost
DB_USER=sa
DB_PWD=(Your database password)
DB_NAME=Flight
DB_PORT=1433
```

* Run in local
```
node index.js
```
